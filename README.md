# panels-blocks

Blocks for panels.

- action
- action with icon
- image
- title
- text
- subtitle

## Install deps

1. Get `iojs`. [Guide](https://gist.github.com/dariocravero/8db219777b277c29fc06).
2. Install the deps `npm install && npm install -g babel-node`.

## Examples

See `playground/playground.es6`.

Run it with `npm start`.
